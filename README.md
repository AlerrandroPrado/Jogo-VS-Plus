## ▶️ Execução do Projeto

Para executar o jogo localmente, siga as instruções abaixo:

1. Faça o download dos arquivos do repositório.
2. Caso necessário, extraia os arquivos compactados (.zip).
3. Acesse a pasta do projeto.
4. Execute o arquivo `VS Plus.exe`.

Após a execução, o jogo será iniciado automaticamente.
